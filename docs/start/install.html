Installation & Configuration

Web Atoms require jQuery 1.7 onwards in your page before including any of following resources. Since jQuery is licensed separately and we are not associated with jQuery or its developers, we request you to download jQuery from their website and include them in your page.

You just need to add three resources in your HTML page.

  1. Any jQuery above 1.7 of your choice
  2. atoms.js
  3. atoms.css - Please note, you must reference atoms.css from within the style folder itself as it references images from the same folder.

You have to add atoms JavaScript and CSS in your page.
